# 餐厅检查评分测算系统

一个现代化的餐厅检查评分测算系统，帮助餐饮行业管理者快速评估餐厅的检查表现，通过标准化的数据输入和精准的计算公式，自动生成餐厅的综合评分、排名和整改优先级。

## 功能特点

- **基础设置区**：可自定义检查覆盖目标值和问题权重
- **数据输入区**：仅需填写6项关键数据（餐厅名称、档口数、检查次数、问题数量等）
- **自动计算区**：根据精准公式自动计算3项核心指标
- **结果输出区**：自动生成餐厅排名和整改优先级标识（红黄绿三色区分）
- **可视化区**：通过柱状图和散点图直观展示餐厅表现
- **数据导出**：支持将所有餐厅数据导出为CSV格式

## 两种主题风格

本系统提供两种视觉风格，可根据个人喜好或使用环境选择：

### 1. 浅色主题 (index.html)

- 简洁明亮的界面设计
- 适合日常办公环境使用
- 经典的白色背景配合蓝色主题色

### 2. 深色数据看板 (index-dark.html)

- 现代化的深色数据看板风格
- 减轻长时间查看数据时的视觉疲劳
- 科技感强，适合专业数据分析场景
- 包含粒子动效背景和网格纹理

## 计算公式

系统使用以下公式进行评分计算：

1. **餐厅检查覆盖充分度** = 检查次数 ÷ 档口数
2. **餐厅问题严重度** = (一般问题×权重1 + 较重问题×权重2 + 严重问题×权重3) ÷ 档口数
3. **综合评分** = (问题严重度 × 0.7) + (1 - 覆盖充分度 ÷ 目标值) × 0.3

## 整改优先级说明

- **红色（高优先级）**：综合评分 ≥ 2分
- **黄色（中优先级）**：综合评分 1-2分
- **绿色（低优先级）**：综合评分 < 1分

## 如何使用

1. 打开 `index.html`（浅色主题）或 `index-dark.html`（深色数据看板）
2. 在"基础设置"区域设置检查覆盖目标值和问题权重
3. 在"数据输入"区域填写餐厅信息和检查数据
4. 点击"计算评分"按钮查看计算结果
5. 点击"添加到列表"按钮将餐厅添加到排名列表
6. 在"结果输出"和"可视化"区域查看分析结果

## 本地运行

1. 克隆或下载本项目到本地
2. 直接在浏览器中打开 `index.html` 或 `index-dark.html` 文件
3. 无需安装任何依赖，所有资源通过CDN加载

## 数据存储

系统使用浏览器的本地存储来保存数据，刷新页面后数据不会丢失。如需清除所有数据，请点击"重置所有"按钮。

## 导出数据

点击顶部的"导出数据"按钮，可将所有餐厅数据导出为CSV格式，方便在Excel等工具中进一步分析。
